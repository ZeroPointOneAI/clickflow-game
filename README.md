# ClickFlow - AI Idle Game

An addictive incremental clicker game where you train an AI brain.

## How to Play

1. Click the brain to earn tokens
2. Buy upgrades to earn passive income
3. Watch your AI grow!

## Play Online

**Live at:** https://zeropointoneai.github.io/clickflow-game/

## Development

Built with pure HTML/CSS/JS - no dependencies.

To run locally:
```bash
# Clone the repo
git clone https://github.com/ZeroPointOneAI/clickflow-game.git
# Open in browser
open index.html
```

## Features

- 🎮 Addictive clicker gameplay
- 🚀 6 upgrades to unlock
- 💾 Auto-save progress
- 🌙 Dark AI-themed aesthetic
- 📱 Mobile-friendly
